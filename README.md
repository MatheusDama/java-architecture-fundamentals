## 🏛️ Matriz de Conhecimento e Formação
Este repositório consolida as práticas, laboratórios e desafios de código originados de duas grandes frentes de estudo em andamento:

1. **Bootcamp Santander 2026 - AI Java Back-end** *(Digital Innovation One - DIO)*
   * **Foco:** Aplicação prática, integração com ferramentas de IA (Copilotos), desenvolvimento de APIs com Spring Boot e projetos de arquitetura voltados ao mercado financeiro.
2. **Oracle Java Foundations** *(Oracle Academy)*
   * **Foco:** Fundamentos rigorosos da linguagem, tipagem, estruturas de controle e os pilares de baixo nível da Programação Orientada a Objetos (POO).

## 🎯 Diretrizes do Repositório
* **Padrões de Mercado:** Versionamento guiado por boas práticas de engenharia, aplicando *Conventional Commits* e revisões de código.
* **Organização Conceitual:** Divisão de diretórios por tópicos técnicos (e não por plataformas de ensino), otimizando a legibilidade para Tech Leads e recrutadores.
* **Aprendizado e práticas limpas:** Limpeza e aplicação do conhecimento de forma sucinta, buscando
resultado e aperfeiçoamento conforme meu desenvolvimento profissional.
